## 熟练使用MarkDown编辑器
- Markdown是一种轻量级标记语言，设计用于简化文本格式化和排版的过程。不管是我们看的官方文档，还是GitHub上的README文件，都是由MarkDown书写的。MarkDown是我们开发人员之间进行信息交流的重要载体
- 推荐的MarkDown编辑器：[Obsidian](https://obsidian.md/)，[Typora](https://typoraio.cn/)
